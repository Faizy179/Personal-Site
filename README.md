# My Personal Portfolio
Welcome to my personal protfolio. I decided to make this website serve as a portfolio rather than just a "personal site". This website serves as a way to show my coding and hardware knowledge in an interactive way.
Check out the website here: https://faizy179.github.io/Personal-Site/

## Features
**Navigation bar:** Navigation bar stays with yoo throughout the entire website so you can easily go from page to page
**Smooth Scrolling:** Clean UI?UX when navigating between pages

## Tech Stack
* **HTML** Used for the website schematic and layout 
* **CSS:** Used to style the website

## How To Run Locally
1. Clone this repository: `git clone https://github.com/Faizy179/Personal-Site.git`
2. Open the project folder
3. Double click on the file named `index.html` to run it in your browser
